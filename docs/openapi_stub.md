# Maestro AI API Reference (Stub)

This is a placeholder for a future OpenAPI/endpoint reference for external integrators.

## Planned endpoints
- `/label/api/mission` — Run mission (POST, auth required)
- `/studio/api/run` — Run studio agent (POST, auth required)
- `/live/api/run` — Run live agent (POST, auth required)
- `/webhook` — Receive webhook (POST, secret required)
- `/platform_ops/api/settings` — Platform settings (GET/POST, admin only)
