How it works

Connect your roster → run your agents → automate your operations

Maestro AI is a modular Flask platform with departments and agents. You can run it fully local (Ollama), integrate cloud LLMs, or scale into a managed deployment later.

1

Connect your roster

Add artists, sessions, shows, tours, and your operating rules—then keep everything in one place.

2

Run your agents

Spin up BRIDGE, VINYL, ECHO, ROUTE, SETTLE, and more from the dashboard or CLI.

3

Automate the work

Review structured outputs, apply changes explicitly, and build repeatable workflows across departments.

Premium Agents

Open core + premium: ship what you need

Maestro AI ships with four premium agents—LEDGER, SAGE Daily Brief, FOCUS, and MULTI_LABEL_ONBOARDING—gated by a single env flag. Everything else is open-source and free to self-host.

1

Enable premium in .env

Set PREMIUM_FEATURES_ENABLED=true in your .env file (copy from .env.example).

2

Open the dashboard

Run python dashboard/app.py, log in, then go to /agents — premium agent pages are linked there.

3

Run and review output

Each agent page has a form, runs server-side, and displays structured results. Outputs are also saved to disk for audit trail.

📖 Full setup guide QUICKSTART

See docs/quickstart.md for full setup instructions including env configuration, LLM provider options (Ollama / Anthropic / OpenAI), running agents via CLI or dashboard, Docker, and the open-core vs premium feature table.

# .env — enable premium agents
PREMIUM_FEATURES_ENABLED=true
MAESTRO_TOKEN=your-secret-token
LLM_PROVIDER=ollama # or anthropic / openai
